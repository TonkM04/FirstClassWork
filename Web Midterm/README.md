# Learning to See 
## Demo
This is a photography website called "Learning to See".

It includes:

* A homepage with introduction and selected photos
* A gallery page displaying all images
* A contact form for users to reach out

---

## Code Snippet I Figured Out

I implemented a function where the user click one a picture then the large veriosin will be displayed on another layer.

Speciaclly when  a user clicks an image:

* The image source is copied into a larger image element
* A full screen overlay appears
* Clicking the overlay closes it (hide)

I used JavaScript (getElementById, classList) and CSS (position, display) to build this feature.

---

## Responsive Design

I used a media query at:

max-width: 768px

On smaller screens:

* The gallery changes to 1 column
* The layout becomes vertical
* Text and spacing adjust for mobile
